## 1. 이 도구는 무엇인가요?

**AI 게임 생성기**는 선생님이 수업 시간에 학생들과 함께 사용할 수 있는 **퀴즈 게임 파일을 자동으로 만들어 주는 웹 서비스**입니다.

### 어떤 문제를 해결하나요?

| 기존 방식 | AI 게임 생성기 사용 시 |
|---|---|
| PPT 게임을 직접 만들려면 수십 장의 슬라이드를 일일이 제작 | 주제만 입력하면 **완성된 PPT가 자동 다운로드** |
| 퀴즈 문제를 하나하나 출제하고 오답 선지까지 구성 | AI가 학년 수준에 맞는 **문제·정답·오답을 자동 생성** |
| 다양한 게임 형식을 만들려면 각각 다른 양식 필요 | **9가지 게임 형식** 중 선택만 하면 자동 변환 |
| 학습지(PDF)도 별도 제작 필요 | 게임에 따라 **PPT와 학습지 PDF가 동시에 생성** |

### 어떤 파일이 만들어지나요?

- **PPT 파일 (.pptx)**: 교실 화면에 띄워 학생들과 함께 진행하는 게임용
- **PDF 파일 (.pdf)**: 학생들에게 나눠 주는 활동지·학습지용
- 게임 종류에 따라 PPT만, PDF만, 또는 **둘 다** 생성됩니다.

## 2. 처음 사용할 때 — 시작하기

### 접속하기

웹 브라우저(크롬, 엣지 등)에서 서비스 주소로 접속합니다. 별도의 회원가입이나 로그인 없이 바로 사용할 수 있습니다.

### 최초 설정 (선택사항)

처음 접속하면 기본값으로 바로 사용 가능하지만, 아래 설정을 미리 해두면 더 편리합니다.

1. 화면 왼쪽 사이드바 하단의 **사용자 이름(기본: "선생님")** 을 클릭합니다.
2. 프로필 설정 창에서 다음을 입력합니다:
   - **사용자 이름**: 원하는 이름 (대시보드에 표시됨)
   - **학급 학생 수**: 우리 반 인원수 입력 (일부 게임에서 이 숫자를 기준으로 문제 수가 결정됩니다)
3. **저장** 버튼을 누릅니다.

> 이 정보는 현재 사용 중인 브라우저에 저장되므로, 같은 브라우저로 재접속하면 유지됩니다.

## 3. 게임 만드는 법 — 단계별 안내

### Step 1. 학년 선택

화면 상단의 **"대상 학년"** 드롭다운에서 수업 대상 학년을 선택합니다.

| 선택지 | 대상 |
|---|---|
| 초등 1-2 | 초등학교 1~2학년 |
| 초등 3-4 | 초등학교 3~4학년 |
| 초등 5-6 | 초등학교 5~6학년 |
| 중학교 | 중학교 |
| 고등학교 | 고등학교 |
| 기타 | 그 외 |

AI가 선택된 학년 수준에 맞춰 문제의 난이도와 어휘를 조절합니다.

### Step 2. 학습 주제 입력

**"학습 주제"** 입력란에 단원명이나 핵심 키워드를 입력합니다.

**좋은 예시:**
- `분수의 덧셈과 뺄셈`
- `조선 후기 실학 사상`
- `식물의 한살이`
- `영어 과거시제 불규칙 동사`

**입력 팁:** 너무 넓은 주제(예: "수학")보다는 **구체적인 단원명이나 소주제**를 적을수록 AI가 더 정확한 문제를 만듭니다.

### Step 3. 문제 수 설정

**"문제 수"** 칸에 만들고 싶은 퀴즈 수를 입력합니다. (1~50개)

> **주의:** 일부 게임은 특정 문제 수가 필요합니다. 자세한 내용은 [5. 게임 종류 소개](#5-게임-종류-소개)를 참고하세요.

### Step 4. 상세 요청 사항 입력 (선택)

AI에게 추가로 전달하고 싶은 내용이 있다면 **"상세 요청 사항"** 칸에 적습니다.

**활용 예시:**
- `쉬운 난이도로 출제해주세요`
- `3단원 내용만 다뤄주세요`
- `OX 문제 형식으로 만들어주세요`
- `실생활 예시를 많이 활용해주세요`

이 칸은 비워도 됩니다.

### Step 5. 참고 자료 PDF 업로드 (선택)

교과서나 학습 자료를 AI에게 직접 전달하여 **더 정확한 문제**를 만들고 싶다면:

1. 화면 우측 상단의 **"참고 자료 PDF"** 버튼을 클릭합니다.
2. 파일 선택 창에서 PDF 파일을 선택합니다 (최대 4MB).
3. 업로드가 완료되면 버튼에 파일명이 표시됩니다.
4. 업로드한 파일을 제거하려면 버튼 우측 상단의 **빨간 X** 를 클릭합니다.

> **참고:** 텍스트뿐 아니라 **이미지, 그림, 도표가 포함된 PDF도 AI가 인식**할 수 있습니다. 그래서 교과서 PDF를 그대로 올려도 활용 가치가 높습니다.

### Step 6. 게임 선택

화면 하단의 **게임 카드** 중 원하는 게임을 클릭합니다. 선택된 게임은 테두리가 강조 표시됩니다. 각 카드에는 게임 이름, 간단한 설명, 생성 파일 형식(PPT/PDF)이 표시되어 있습니다.

### Step 7. 생성하기

모든 설정이 끝나면 우측 상단의 **"지금 바로 생성하기"** 버튼을 클릭합니다.

1. 버튼을 누르면 AI가 퀴즈를 생성하기 시작합니다.
2. 화면에 진행 상태 메시지가 표시됩니다:
   - `"AI가 수업 주제를 분석하여 최적의 퀴즈를 생성하고 있습니다..."` → 문제 생성 중
   - `"○○ 파일을 생성하고 있습니다..."` → 파일 제작 중
3. 완료되면 **파일이 자동으로 다운로드**됩니다.
4. 브라우저의 다운로드 폴더에서 파일을 확인하세요.

> **소요 시간:** 보통 10~30초 정도 걸리며, 문제 수가 많거나 PDF 자료를 함께 분석하는 경우 조금 더 걸릴 수 있습니다.

## 5. 게임 종류 소개

### 한눈에 비교

| 게임 | 생성 파일 | 한 줄 설명 | 문제 수 조건 |
|---|:---:|---|---|
| 텔레파시 | PPT | 둘 중 맞는 답을 고르는 이심전심 퀴즈 | 자유 (내부적으로 2배 출제) |
| 뻥이요! | PPT + PDF | 폭탄을 피하며 살아남는 스릴 퀴즈 | 자유 |
| 야구골든벨 OX | PPT | O/X로 판단하는 야구 게임 | 자유 |
| 집어! | PPT | 가장 먼저 정답을 낚아채는 순발력 게임 | 자유 |
| 레벨업 골든벨 | PPT | 단계별 관문을 통과하는 도전형 | 자유 |
| 땅따먹기 | PDF | 보드판 위에서 영토를 넓히는 학습지 | **반드시 48문제** |
| 빙고 | PDF + PPT | 키워드 빙고 활동지 + 진행용 PPT | 자유 |
| 고고 전진 | PDF + PPT | 주사위 보드게임 미션지 + 진행용 PPT | **최소 12문제** |
| 탑텐짝찾기 | PDF | 카드를 뒤집어 짝을 찾는 매칭 게임 | **학생 수 × 2** (아래 참고) |

### 각 게임 상세 설명

#### 텔레파시 (PPT)
- **진행 방식**: 매 라운드마다 2개의 문제가 나오고, 학생은 둘 중 하나를 선택합니다. 선생님이 정한 정답과 같으면 통과!
- **문제 수 참고**: 10라운드를 원하면 문제 수를 10으로 설정합니다. AI가 자동으로 20문제(10라운드 × 2)를 출제합니다.
- **추천 상황**: 전체 학급이 함께 참여하는 간단한 워밍업 활동

#### 뻥이요! (PPT + PDF)
- **진행 방식**: 퀴즈에 답하면서 랜덤 보너스/폭탄 포인트가 섞여 긴장감 있게 진행됩니다.
- **생성 파일**: 게임 진행용 PPT + 학생 활동지 PDF 2개가 함께 생성됩니다.
- **추천 상황**: 모둠 대항전, 스릴 있는 복습 활동

#### 야구골든벨 OX (PPT)
- **진행 방식**: O/X 퀴즈를 통해 야구 경기처럼 진루·아웃을 진행합니다.
- **추천 상황**: O/X 판단이 명확한 주제 (과학 개념, 역사 사실 등)

#### 집어! (PPT)
- **진행 방식**: 화면에 정답이 나타나면 가장 먼저 낚아채는 순발력 게임입니다.
- **추천 상황**: 활동적인 분위기를 원할 때, 단답형 복습

#### 레벨업 골든벨 (PPT)
- **진행 방식**: 레벨 1부터 시작해 관문을 통과하며 점점 어려워지는 단계별 퀴즈입니다.
- **추천 상황**: 수업 정리 시간, 단원 마무리 복습

#### 땅따먹기 (PDF)
- **진행 방식**: 6×8(총 48칸) 보드판 위에서 문제를 맞히며 영토를 확장하는 보드게임 학습지입니다.
- **문제 수 조건**: **반드시 48문제**로 설정해야 합니다. 보드판의 모든 칸을 채워야 하기 때문입니다.
- **추천 상황**: 모둠 활동, 짝 활동

#### 빙고 (PDF + PPT)
- **진행 방식**: 학생이 빙고판에 키워드를 채우고, 선생님이 PPT로 문제를 진행하면서 빙고를 완성합니다.
- **생성 파일**: 학생 빙고 활동지 PDF + 교사 진행용 PPT 2개가 함께 생성됩니다.
- **추천 상황**: 핵심 용어 정리, 단원 복습

#### 고고 전진 (PDF + PPT)
- **진행 방식**: 주사위를 굴려 보드판 위를 전진하며, 멈춘 칸의 미션(퀴즈)을 해결하는 보드게임입니다. 6단계로 구성됩니다.
- **문제 수 조건**: **최소 12문제** 이상 설정해야 합니다.
- **생성 파일**: 미션 학습지 PDF + 진행용 PPT 2개가 함께 생성됩니다.
- **추천 상황**: 모둠별 보드게임 활동

#### 탑텐짝찾기 (PDF)
- **진행 방식**: 카드를 뒤집어 서로 관련 있는 짝을 찾는 매칭 게임입니다.
- **문제 수 조건**: 설정된 학급 학생 수에 따라 결정됩니다.
  - 계산법: **학생 수 × 4 = 총 카드 수**, **총 카드 수 ÷ 2 = 필요한 문제 수**
  - 예: 학생 24명 → 카드 96장 → **문제 수 48개**로 설정
  - 예: 학생 30명 → 카드 120장 → **문제 수 60개**로 설정 (이 경우 최대 50개 제한에 걸릴 수 있으므로 학생 수 조정 필요)
- **추천 상황**: 개념과 정의 매칭, 영어 단어-뜻 짝짓기

## 6. 나의 게임 보관함

사이드바에서 **"나의 게임 보관함"** 을 클릭하면, 이전에 만든 게임 목록을 확인할 수 있습니다.

### 주요 기능

- **최근 20건**의 게임 기록이 자동으로 저장됩니다.
- 각 기록에는 **주제, 게임명, 대상 학년, 생성 일시**가 표시됩니다.
- **다시 다운로드** 버튼을 클릭하면 동일한 파일을 다시 받을 수 있습니다.

### 주의 사항

- 게임 기록은 **현재 브라우저에만** 저장됩니다. 다른 기기나 브라우저에서는 보이지 않습니다.
- 브라우저의 데이터를 삭제(캐시 삭제 등)하면 보관함 기록도 함께 사라집니다.
- 중요한 파일은 다운로드 후 **별도 폴더에 보관**해 두는 것을 권장합니다.

## 7. 프로필 설정

사이드바 하단의 **사용자 이름**을 클릭하면 프로필 설정 창이 나타납니다.

| 설정 항목 | 설명 | 기본값 |
|---|---|---|
| **사용자 이름** | 대시보드에 표시될 이름 | 선생님 |
| **학급 학생 수** | 탑텐짝찾기 등 일부 게임의 카드 수 계산에 사용 | 24명 |
| **Kimi API Key** | 개인 API 키 (선택사항, 아래 설명 참고) | 비어 있음 |

### Kimi API Key란?

이 서비스는 AI 퀴즈 생성을 위해 **Kimi API**를 사용합니다. 기본적으로 공용 키가 제공되어 바로 사용할 수 있지만, 동시 사용자가 많으면 일시적으로 오류가 발생할 수 있습니다.

이런 경우 **개인 API 키**를 발급받아 등록하면 안정적으로 사용할 수 있습니다.

> 개인 키 발급은 [Moonshot AI 공식 사이트](https://platform.moonshot.cn/)에서 가능합니다. 일반적으로는 공용 키로 충분하므로, 오류가 반복되는 경우에만 고려하시면 됩니다.

## 8. 자주 묻는 질문 (FAQ)

### Q. 파일이 다운로드되지 않아요.
**A.** 브라우저가 팝업이나 자동 다운로드를 차단하고 있을 수 있습니다. 브라우저 주소 표시줄 오른쪽에 차단 알림이 있는지 확인하고, "허용"으로 변경해 주세요.

### Q. 생성 버튼이 눌리지 않아요.
**A.** **"학습 주제"** 가 비어 있으면 버튼이 비활성화됩니다. 주제를 입력했는지 확인해 주세요.

### Q. PPT 파일이 열리지 않아요.
**A.** 이 서비스는 **.pptx** 형식의 파일을 생성합니다. Microsoft PowerPoint, Google Slides, 한컴오피스 등에서 열 수 있습니다. 모바일에서는 해당 앱이 필요합니다.

### Q. PDF가 아닌 한글(hwp) 파일을 올릴 수 있나요?
**A.** 현재는 **PDF 파일만** 지원됩니다. 한글 파일은 "다른 이름으로 저장" → "PDF"로 변환한 뒤 업로드해 주세요.

### Q. 생성된 문제의 질이 마음에 들지 않아요.
**A.** 다음을 시도해 보세요:
- **주제를 더 구체적으로** 입력 (예: "수학" → "5학년 약분과 통분")
- **상세 요청 사항**에 추가 지침 작성 (예: "교과서 125~130쪽 내용 위주로")
- **참고 자료 PDF**를 업로드하여 AI가 정확한 내용을 참고하도록 유도

### Q. 같은 주제로 다시 만들면 같은 문제가 나오나요?
**A.** 아닙니다. AI가 매번 새롭게 문제를 생성하므로 **다른 문제**가 출제됩니다.

### Q. 만든 게임을 수정할 수 있나요?
**A.** 다운로드된 PPT 파일은 일반 PowerPoint 파일이므로, **PowerPoint에서 자유롭게 수정**할 수 있습니다. 문제를 바꾸거나, 디자인을 변경하거나, 슬라이드를 추가/삭제할 수 있습니다.

### Q. 인터넷이 없으면 사용할 수 없나요?
**A.** AI 퀴즈 생성을 위해 인터넷 연결이 필요합니다. 단, 보관함에서 이전에 만든 게임을 다시 다운로드하는 것은 가능합니다.

## 9. 문제 해결 (트러블슈팅)

| 증상 | 원인 | 해결 방법 |
|---|---|---|
| `"생성 중 오류 발생"` 메시지 | AI 서버 과부하 또는 API 키 한도 초과 | 잠시 후 다시 시도하거나, 프로필 설정에서 개인 API 키 등록 |
| `"48개의 문제가 필요합니다"` 알림 | 땅따먹기 게임의 필수 문제 수 미충족 | 문제 수를 **48**로 변경 |
| `"최소 12개 이상"` 알림 | 고고 전진 게임의 최소 문제 수 미충족 | 문제 수를 **12 이상**으로 변경 |
| `"카드 수가 학생 수×4여야 합니다"` 알림 | 탑텐짝찾기의 문제 수 불일치 | 알림에 표시된 숫자대로 문제 수 변경 |
| `"PDF 파일만 업로드 가능합니다"` | PDF가 아닌 파일 업로드 시도 | .pdf 파일만 선택하여 업로드 |
| `"4MB 이하의 파일만"` | PDF 용량 초과 | 파일 용량을 줄이거나 페이지를 나누어 업로드 |
| 파일은 생성되었으나 내용이 깨져 보임 | PPT 뷰어 호환성 문제 | Microsoft PowerPoint 또는 Google Slides로 열기 |
| 로딩이 1분 이상 지속 | 네트워크 문제 또는 서버 지연 | 페이지 새로고침 후 다시 시도 |
